XXE, is it possible to automate?

In this series I want to analyze the most simple Portswigger excercises for the XXE injection vulnerability (link here) and try to think, while going through them, if the check for this vulnerability can be automated via a simple Python script. The idea is to start with a very basic version of the script that searches for the most simple XXE injection case and then refactor the script and modify it to work for new and more complex cases.
